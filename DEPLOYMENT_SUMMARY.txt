================================================================================

1. Read: PRODUCTION_PERMISSIONS_FIX.md

2. Deploy using Option A (Recommended):
   
   cd /var/www

   cd rangamanch
   composer install --optimize-autoloader --no-dev

   cp .env.production .env

   # Deploy
   php artisan key:generate
   php artisan migrate --force

================================================================================
IF YOU GET "Permission denied" ERROR ON PRODUCTION
================================================================================

See: PRODUCTION_PERMISSIONS_FIX.md

Quick Fix:
1. SSH to your server
2. cd /var/www/rangamanch (or your path)
3. Run:
   sudo chown -R www-data:www-data storage bootstrap/cache
   sudo chmod -R 755 storage bootstrap/cache
   
   (Replace 'www-data' if your server uses 'nginx', 'daemon', etc.)

4. Try the artisan command again
